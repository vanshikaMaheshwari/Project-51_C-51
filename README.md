# Project-51_C-51 — Disney Princess Qualities

A small web project exploring Disney princess characters and their qualities, built with HTML, CSS, and p5.js (sketch.js).

## Pages
- `index.html` — homepage
- `qualities.html` — princess qualities page
- `aboutme.html` — about the creator
- `followme.html` — social links

## Running locally
1. Clone the repo
2. Open `index.html` in your browser
